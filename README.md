# Marketus

### 프로젝트 특징
 - 빅데이터와 딥러닝 기반의 인플루언서 분석 및 추천 서비스 프로젝트

### 서비스 흐름도

### 사용한 딥러닝 모델
- GRU Model : 포스팅 카테고리 분류
- LSTM Model : 광고 댓글 분류 & 개시물 댓글 진짜/가짜 분류
- MLP Model : 인플루언서 팔로워 진짜/가짜 분류
- GloVe Model : 해시태그 유사도 분석 및 추천
